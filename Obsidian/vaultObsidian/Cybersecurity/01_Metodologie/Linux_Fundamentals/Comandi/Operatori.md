**Gli operatori permettono di aggiungere funzionalità ai comandi da terminale**

| Simbolo Operatore | Descrizione                                                                                                                                                                                                    |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| &                 | **Permette di far runnare il comando in background**.<br>Molto utile ad esempio su server così da poter eseguire altri comandi senza bloccare la schermata del terminale.                                      |
| &&                | **Permette di concatenare più comandi tra loro**, il secondo viene eseguito solo se il primo ha successo                                                                                                       |
| >                 | **Permette di reindirizzare un output verso ad esempio un file**.<br>Di default il contenuto del file viene automaticamente sovrascritto con il contenuto dell'output.<br>Se il file non esiste, viene creato. |
| >>                | Come il comando precedente, la sola differenza è che **il contenuto non viene sovrascritto** ma aggiunto con delle nuove righe.                                                                                |
