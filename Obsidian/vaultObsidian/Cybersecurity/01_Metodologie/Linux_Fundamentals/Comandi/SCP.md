**Permette di scaricare o caricare file mediante un collegamento SSH (Secure Shell)**

Download di un file
```bash
scp nomeutente@ipaddress:/percorso/file.txt output_file.txt
```

Upload di un file
```bash
scp output_file.txt nomeutente@ipaddress:/percorso/file.txt 
```
